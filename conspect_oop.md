## 1. Указатели (Pointers)

Указатель — переменная, хранящая **адрес памяти** другой переменной.

### Синтаксис
```cpp
int a = 42;
int *p = &a;   // p = адрес a
cout << *p;    // разыменование -> 42
*p = 100;      // a теперь 100
